Outdork


An android application that encourages people to be more outdoors, it gives examples of tracks and other outdoor activities to do. Outdork was a school project where I was a part in a team of 4 students. It was written in Kotlin using Android Studio and Firebase for storing.
Press on Outdork to find it on GitHub.

Zetkin


Zetkin foundation is an organization that provides a tool for organizational activism and helps organizations with organizing memberships, events and activities. I have been volunteering to help with the frontend development of the platform for about 2 years on my free time.
Read more about Zetkin by pressing the title.

Tetris


A fun interpretation of the classical archade game Tetris. Made in vanilla Javascript.

World Clocks


A world clock application that lets you see what time it is arounf the world. You can also add you own city and choose what time zone that city belongs to.
Test the application by pressing on the title World Clocks.

Asteria


A full stack web application made as a school project. Developed with Javascript, Node and Express, with an SQLite database. It is an application made for booking tickets, viewing trailers and information about movies, and handling your own bookings. The team consisted of 5 students.
Check out the source code on GitHub by clicking on the name Asteria.
